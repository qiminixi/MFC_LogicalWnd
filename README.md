# MFC_LogicalWnd
A simple framework of DUI(Direct UI).    
What is in it:  
* Logical window, it is not window of OS.  
* window hierarchy and focous, focoused window is in front.  
* window messages, like resize, mousebutton, keyboard and so on.  
* button control.  
* gif player.
